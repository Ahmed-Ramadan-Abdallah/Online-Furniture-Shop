# Online Furniture Shop
The online furniture store showcases various rooms in a house and offers multiple features. For instance, it filters furniture based on different rooms. Upon selecting specific furniture, individual product information is displayed. Verify product availability and, if the item is in stock, add it to the shopping cart.

  <h4>Technologies used:</h4>
  <li>React</li>
  <li> Redux</li>
  <li>React Router Dom</li>
  <li>Context API</li>
  <li>Styled Component</li>
</ul>
